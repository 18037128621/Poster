# 海报生成器

使用 &lt;canvas&gt; 的**[海报生成器](http://www.geekbang.org/poster)**。

*注：页面主要为手机端设计，在电脑浏览器中打开时，请缩小浏览器宽高比至手机竖屏比例。*

## 构建

React + ES6 + SCSS + Webpack
